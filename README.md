## Interactive 3D City Map – Dexhigh Assessment

### Tech Stack
- Next.js
- React Three Fiber / Three.js
- Tailwind CSS

### Features
- 6-direction snap rotation (60°)
- Hexagonal compass navigation
- Marker interaction with detail cards
- Fully responsive (desktop & mobile)
- Optimized 3D assets

### Controls
- Use compass or Previous/Next buttons to rotate
- Click markers to view details

### Performance
- Optimized meshes
- Smooth transitions
